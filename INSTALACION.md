# 📲 Guía de Instalación - Nexo Science Search

**Un solo repositorio para todas las plataformas.**

Esta es una Progressive Web App (PWA) que puedes instalar como aplicación real en **iOS, Android, Windows, macOS y Linux** sin pagar licencias de desarrollador.

## Plataformas soportadas

| Plataforma     | Guía de instalación                  | Método recomendado      |
|----------------|-------------------------------------------|-----------------------------|
| iPhone / iPad  | [INSTALAR_IOS.md](INSTALAR_IOS.md)        | Archivo HTML + Safari       |
| Android        | [INSTALAR_ANDROID.md](INSTALAR_ANDROID.md)| APK o versión Web (PWA)   |
| Windows        | [INSTALAR_WINDOWS.md](INSTALAR_WINDOWS.md)| Versión Web (HTML)        |
| macOS          | [INSTALAR_MAC.md](INSTALAR_MAC.md)        | Versión Web (HTML)        |
| Linux          | [INSTALAR_LINUX.md](INSTALAR_LINUX.md)    | Versión Web (HTML)        |

## ⚡ Ventajas

- Totalmente gratis
- No requiere publicar en tiendas
- Funciona sin conexión después de la primera búsqueda
- Un solo lugar centralizado
