# 🍎 Instalar Nexo Science Search en iPhone o iPad (usando el archivo HTML)

**No necesitas pagar la licencia de Apple Developer.**

La versión web viene como un archivo `.html` que puedes instalar fácilmente como aplicación.

## Pasos para instalar desde Google Drive:

1. Entra a la carpeta de Google Drive:
   🔗 [Carpeta Google Drive](https://drive.google.com/drive/folders/1lW-jw7mz9MaHk-HAiV81uoUrafzNUi3R)

2. Descarga el archivo llamado **`nexo-science-search.html`**

3. Toca el archivo descargado y elige **"Abrir en Safari"**

4. Una vez que se abra la página en Safari:
   - Toca el botón **Compartir** ⬆️ (abajo de la pantalla)
   - Desplaza hacia abajo y selecciona **"Agregar a pantalla de inicio"**
   - (Opcional) Cambia el nombre
   - Toca **Agregar**

## ✅ ¡Listo!

Ahora tendrás **Nexo Science Search** como una aplicación real en tu pantalla de inicio, con su propio icono.

### Qué pasa después:
- Se abre en **pantalla completa** (sin barra de Safari)
- Funciona como una app nativa
- La primera vez necesita internet para buscar artículos

> 💡 Este método es el más recomendado para usuarios de iPhone/iPad.
